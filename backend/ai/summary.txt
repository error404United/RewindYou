Dynamic Programming is a commonly used algorithmic technique used to optimize recursive solutions when same subproblems are called again. The core idea behind DP is to store solutions to subproblems so that each is solved only once. To make sure that a recursive value is computed only once (to improve time taken by algorithm), we store results of the recursive calls. There are two ways to store the results, one is top down (or memoization) and other is bottom up (or tabulation).
